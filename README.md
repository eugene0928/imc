## IMC project

### Prerequisites 
- Nodejs: <= 20.0
- Postgres: 14.9


## Installation

### Intall dependencies
```bash
npm install
```

### Create `.env` file for credentials
```bash
touch .env
```

### Update `.env` file. Add `DATABASE_URL` key with proper value

### Run on watch mode
```bash
npm run start:dev
```


### Run project
```bash
npm start
```

### Run unit tests
```bash
npm run test
```
