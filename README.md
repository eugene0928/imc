## IMC project

### Prerequisites 
- Nodejs: >= 20.0
- Postgres: 14.9

### Pacakage manager is `YARN`


## Installation

### Intall dependencies
```bash
yarn install
```

### Create `.env` file for credentials
```bash
touch .env
```

### Update `.env` file. Add `DATABASE_URL` key with proper value

### Run on watch mode
```bash
yarn run start:dev
```


### Run project
```bash
yarn start
```

### Run unit tests
```bash
yarn run test
```
